Primary Predictions

Using machine learning to predict the outcome of the democratic primaries by county.

Created by Terra Thompson & John Loper

Sources: Washington Post

View Github

Using a random forest classifier model, we predicted whether Democratic 2020 candidates Bernie Sanders and Joe Biden would win or lose counties.

A few notes:

  • The model uses the results up to and including Super Tuesday
  • Since we predict the candidates separately, they can both win or lose the same county
  • We did not predict states that were used in the model
